    poi

    poi

    Contains constants and functions for a POI Object.

    The structure for a POI Object is as follows:

    {
  latitude: <number>,
  libraryid: <string>,
  longitude: <number>,
  regionname: <string>,
  regionmetadata: {
    category: <string>,
    city: <string>,
    country: <string>,
    state: <string>,
    street: <string>,
  },
  radius: <number>,
  regionid: <string>,
  useriswithin: <boolean>,
  weight: <number>,
  type: <string>,
  uuid: <string>,
}
